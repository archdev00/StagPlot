# StagPlot
Python scripts for plotting fields from a StagYY output using the StagPy backend.

## First steps:
Create a virtual environment for StagPy in your preferred directory with this command (this is crucial to avoid dependency or library conflicts) :\
`python3 -m venv myenv`

Now activate the environment (Linux):\
`source myenv/bin/activate`

Install StagPy:\
`pip install stagpy`

If you are using VSCode, open the directory with it. Press ctrl+shift+p (or type ">" in the search box) and select the Python Interpreter as "myenv".

Now you can run the scrips using the virtual environment you created.
